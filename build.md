# seedsui-react

## 工程构建

初始化

```bash
npx @umijs/create-dumi-lib --site
```

使用移动模板

```bash
npm install dumi-theme-mobile  -D -f
```

## 开始

安装依赖

```bash
$ npm i
```

启用服务

```bash
$ npm start
```

编译文档

```bash
$ npm run docs:build
```

启动测试

```bash
$ npm test
```

编译 library via `father`,

```bash
$ npm run build
```
